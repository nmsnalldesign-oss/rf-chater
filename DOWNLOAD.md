# Download Full Code

All complete source code files are provided in this repository.

## Option 1: Clone with Git (Recommended)

```bash
git clone https://github.com/nmsnalldesign-oss/rf-chater.git
cd rf-chater
```

This downloads everything including hidden git files.

## Option 2: Download ZIP

1. Click green `Code` button on main repo page
2. Select `Download ZIP`
3. Extract the archive
4. Open terminal in extracted folder
